GitHubのコミットログ一覧を表示する

Hi 👋

I made a simple tool to generate an image of all your Github contributions! So you can tweet it ✨

Check it out here: https://t.co/boeScBUOza

Disclaimer: It’s still very raw. Just wanted something for myself, but feel free to send PRs and submit issues ♥️ pic.twitter.com/wgVLK7t0w4

— Sallar 👨🏻‍💻 (@sallar) 2018年5月1日

GitHubのコミットログ一覧を表示するGitHub Contributions Chart Generatorです。

2015年の終わり頃から、GitHubに毎日コミットしようという自分ルールを作って実践してきたので、ログを作って見ました。

毎日コミットすることはできませんでしたが、けっこう草を生やすことができました！

以前から活動していたチーム活動がなくなってしまいプライベートリポジトリをGitLabに移行したので、記念に作っておきました。 現在私のアカウントに草は生えておりません。

会社でGitHubを使用するようになったので、個人ではパブリックなリポジトリへの貢献を増やしていきたいと思います。
